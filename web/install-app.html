Get SIP&GO! as a native app on any device. Works offline, loads instantly, and feels just like a regular app!

🚀 Quick Install

Most modern browsers will show an install prompt automatically. Look for the install icon in your address bar!

📱 Go to App & Install

📱 Install on iPhone/iPad

1

Open Safari Browser

Visit sipandgo.app using Safari (not Chrome or other browsers)

💡 Tip: PWA installation only works with Safari on iOS devices
2

Tap the Share Button

Look for the share icon (square with arrow pointing up) at the bottom of Safari

📍 Location: Bottom center of Safari browser
3

Select "Add to Home Screen"

Scroll down in the share menu and tap "Add to Home Screen"

🔍 Look for the plus (+) icon next to this option
4

Confirm Installation

Tap "Add" in the top right corner. SIP&GO! will appear on your home screen!

🎉 Success! You can now launch SIP&GO! like any other app

🤖 Install on Android

1

Open Chrome Browser

Visit sipandgo.app using Chrome browser on your Android device

💡 Tip: Chrome usually shows an automatic install prompt
2

Look for Install Prompt

Chrome will show a banner asking "Add SIP&GO! to Home screen?" - tap "Add"

🔔 If you don't see it, try the menu method below
3

Alternative: Use Chrome Menu

Tap the three dots (⋮) menu → "Add to Home screen" → "Add"

📋 This method works if the automatic prompt doesn't appear
4

Launch from Home Screen

Find the SIP&GO! icon on your home screen and tap to launch!

⚡ The app will load instantly and work offline

💻 Install on Desktop

1

Open Chrome, Edge, or Firefox

Visit sipandgo.app in your desktop browser

🌐 Works with most modern browsers
2

Look for Install Icon

Check your address bar for a computer/install icon - click it

📍 Usually appears on the right side of the address bar
3

Click "Install"

A dialog will appear asking to install SIP&GO! - click "Install"

⚡ Installation takes just a few seconds
4

Launch from Desktop

SIP&GO! will appear in your apps menu and desktop. Launch like any native app!

🚀 Runs in its own window, no browser tabs needed
