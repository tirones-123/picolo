Install SIP&GO! App

Get the full app experience! Install SIP&GO! on your device for faster access, offline play, and a native app feel. No app store required - install directly from your browser.

Install on iPhone/iPad (iOS)

1

Open in Safari

Visit sipandgo.app in Safari browser (this won't work in Chrome or other browsers on iOS).

2

Tap the Share Button

Look for the share icon at the bottom of your screen (square with arrow pointing up).

3

Select "Add to Home Screen"

Scroll down in the share menu and tap "Add to Home Screen" option.

4

Confirm Installation

You can change the app name if desired, then tap "Add" to install SIP&GO! to your home screen.

Need Help?

If you're having trouble installing the app, you can always use SIP&GO! directly in your browser. The web version works perfectly and includes all the same features!

Troubleshooting Tips:

  • Make sure you're using a supported browser (Chrome, Safari, Edge, Firefox)
  • Check that your browser is up to date
  • Try refreshing the page if the install option doesn't appear
